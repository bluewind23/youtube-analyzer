YouTube API 키 설정 가이드

YouTube 분석 서비스를 이용하기 위한 API 키 생성부터 등록까지 단계별로 안내해드립니다.

소요 시간: 약 5-10분 | 필요한 것: 구글 계정 | 비용: 무료

1

Google Cloud Console 접속

구글 클라우드 콘솔에 접속하여 새 프로젝트를 생성합니다.

🔗 접속 링크:

https://console.cloud.google.com/

✅ 체크포인트:

  • • 구글 계정으로 로그인 완료
  • • Google Cloud Console 대시보드 표시
  • • 프로젝트 선택 드롭다운 확인

💡 팁:

처음 접속하시는 경우 서비스 약관 동의가 필요할 수 있습니다.

2

새 프로젝트 생성

YouTube API 키를 위한 새로운 프로젝트를 생성합니다.

주의사항: 기존 프로젝트가 있다면 새로 만들거나 기존 프로젝트를 사용할 수 있습니다.

📋 단계별 진행:

  1. 화면 상단의 "프로젝트 선택" 드롭다운 클릭
  2. "새 프로젝트" 버튼 클릭
  3. 프로젝트 이름 입력 (예: "YouTube-Analyzer")
  4. "만들기" 버튼 클릭
  5. 프로젝트 생성 완료까지 잠시 대기
3

YouTube Data API v3 활성화

생성된 프로젝트에서 YouTube Data API v3를 활성화합니다.

🚀 빠른 링크:

YouTube Data API v3 페이지로 바로 이동

📋 단계별 진행:

  1. 왼쪽 메뉴에서 "API 및 서비스" → "라이브러리" 클릭
  2. 검색창에 "YouTube Data API v3" 입력
  3. 검색 결과에서 "YouTube Data API v3" 클릭
  4. "사용 설정" 버튼 클릭
  5. API 활성화 완료까지 잠시 대기
4

API 키 생성

YouTube API에 접근하기 위한 API 키를 생성합니다.

📋 단계별 진행:

  1. 왼쪽 메뉴에서 "API 및 서비스" → "사용자 인증 정보" 클릭
  2. 상단의 "+ 사용자 인증 정보 만들기" 클릭
  3. "API 키" 선택
  4. API 키가 자동으로 생성됩니다
  5. "키 제한" 버튼을 클릭하여 보안 설정 (권장)

보안 주의: API 키는 절대 다른 사람과 공유하지 마세요. 이 키로 YouTube API를 사용하면 요금이 발생할 수 있습니다.

5

API 키 제한 설정 (권장)

보안을 위해 API 키 사용 범위를 제한합니다.

📋 API 제한 설정:

  1. "API 제한" 섹션에서 "키 제한" 선택
  2. "YouTube Data API v3" 체크박스 선택
  3. 다른 API는 모두 해제
  4. "저장" 버튼 클릭

💡 선택사항 - 애플리케이션 제한:

더 높은 보안을 원한다면 "애플리케이션 제한" 섹션에서 "HTTP 리퍼러(웹사이트)"를 선택하고 사용할 도메인을 추가할 수 있습니다.

6

API 키 등록

생성된 API 키를 YouTube 분석 서비스에 등록합니다.

📋 등록 방법:

  1. Google Cloud Console에서 생성된 API 키 복사
  2. {% if current_user.is_authenticated %}마이페이지{% else %}로그인 후 마이페이지{% endif %}로 이동
  3. "API 키 관리" 섹션에서 "새 API 키 추가" 클릭
  4. 복사한 API 키를 입력란에 붙여넣기
  5. "추가" 버튼 클릭

축하합니다! 설정이 완료되었습니다.

이제 YouTube 분석 서비스를 자유롭게 이용하실 수 있습니다.

자주 묻는 질문 (FAQ)

Q. API 키 사용에 비용이 발생하나요?

YouTube Data API v3는 일일 할당량 내에서 무료로 사용할 수 있습니다. 기본 할당량은 일반적인 사용에 충분하며, 초과 시에만 요금이 발생합니다.

Q. API 키가 작동하지 않아요.

API 키 생성 후 활성화까지 몇 분이 소요될 수 있습니다. 잠시 기다린 후 다시 시도해보시고, 여전히 문제가 있다면 API 키 제한 설정을 확인해주세요.

Q. 여러 개의 API 키를 등록할 수 있나요?

네, 여러 개의 API 키를 등록할 수 있습니다. 시스템이 자동으로 사용 가능한 키를 선택하여 안정적인 서비스를 제공합니다.

Q. API 키를 잘못 입력했어요.

마이페이지에서 등록된 API 키를 삭제하고 새로운 키를 다시 등록할 수 있습니다. 또는 Google Cloud Console에서 새 API 키를 생성하여 등록하세요.
